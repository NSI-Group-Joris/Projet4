Tim Berners-Lee


Un grand homme derrière un grand projet

Tim Berners-Lee, né le 8 juin 1955 à Londres, est un physicien et informaticien britannique, principal inventeur du World Wide Web (WWW, le Web) alors qu'il travaillait à l'Organisation européenne pour la recherche nucléaire (CERN) au tournant des années 1990. Depuis 1994, il préside le World Wide Web Consortium (W3C), organisme qu'il a fondé. En juillet 2004, il est fait chevalier par la reine Élisabeth II. Il est lauréat du prix Turing 20161. En créant le Web, Tim Berners-Lee invente ses trois technologies fondatrices : les adresses Web sous forme d'URL, le protocole de communication HTTP, et le langage informatique HTML. Il fut aidé à ses débuts par l'ingénieur et informaticien belge Robert Cailliau qui cosigna notamment avec lui, en novembre 1990, un document désormais entré dans l'Histoire et intitulé « WorldWideWeb : Proposition pour un projet hypertexte.

A quoi sert le language HTML ?

HTML (HyperText Markup Language) est un langage de balisage utilisé pour structurer le contenu d'une page web. Il permet de définir les éléments de base d'une page, comme les titres, paragraphes, images, liens, et formulaires, en les encapsulant dans des balises spécifiques. Bien qu'il ne s'occupe pas de la mise en forme détaillée, HTML fournit la structure essentielle à laquelle des technologies comme CSS et JavaScript viennent s'ajouter pour améliorer l'apparence et l'interactivité de la page. En résumé, HTML est indispensable pour créer la base de tout site web.


Comment fonctionne le HTML ?

  • Le fonctionnement de HTML repose sur l'utilisation de balises pour structurer et organiser le contenu d'une page web. Lorsqu'un navigateur web charge une page HTML, il interprète ces balises pour afficher le contenu de manière visuelle.
  • Un document HTML commence par la déclaration "!DOCTYPE html", suivie de la balise "html", qui englobe tout le contenu. À l'intérieur, il y a généralement deux sections : l'en-tête "head" où l'on place des métadonnées, le titre de la page ou des liens vers des ressources externes comme le CSS, et le corps "body" où se trouve le contenu visible, comme les textes, images, vidéos et autres éléments interactifs. HTML organise ainsi le contenu d'une page, mais pour la mise en forme et les interactions, on utilise généralement CSS et JavaScript en complément.
