D'où vient le CSS ?

Le CSS (Cascading Style Sheets) a été créé par Håkon Wium Lie en 1994, alors qu'il était employé au CERN , l'organisation européenne pour la recherche nucléaire (est l'un des environnements informatiques scientifiques les plus exigeants du monde).

L'idée derrière le CSS était de séparer la présentation visuelle des pages web (couleurs, polices, mises en page) de leur structure HTML Cela permettait d'améliorer l'accessibilité, la maintenance du code et de rendre les pages plus flexibles.

Le concept de feuille de style est présent depuis l'origine du World Wide Web : le premier navigateur web WorldWideWeb.

La première spécification officielle a été publiée en 1996 par le W3C (World Wide Web Consortium).


À quoi sert-il ?

Le CSS permet de contrôler l’apparence des éléments HTML, comme les couleurs, les polices, les espacements, les bordures, les mises en page et même les animations. Il vient compléter le HTML en offrant une grande flexibilité pour styliser les pages web.

Grâce au CSS, on peut créer des designs cohérents et responsives, adaptés à différents appareils, qu’il s’agisse de bureaux, de tablettes ou de smartphones.

Il permet de séparer le contenu HTML de la présentation, facilitant ainsi la maintenance du code et améliorant l'accessibilité du site.

En résumé, le CSS rend les sites web non seulement esthétiques, mais aussi fonctionnels et parfaitement adaptés aux besoins des utilisateurs.


Comment fonctionne-t-il ?

Le CSS fonctionne en définissant des règles de style qui s'appliquent aux éléments HTML d'une page web. Voici comment il fonctionne concrètement :

  • En CSS, les sélecteurs sont utilisés pour choisir (ou cibler) les éléments HTML auxquels on veut appliquer des styles. En d'autres termes, les sélecteurs permettent de dire "je veux changer ceci" dans la page web.
  • Propriétés et valeurs : Une fois un élément ciblé, on lui attribue des propriétés, comme la couleur, la taille de la police, l'espacement, etc. Chaque propriété a une valeur. Par exemple : color: red; ou font-size: 16px;.
